Descrição

Site responsivo, feito com base em um desafio do FrontEnd Mentor, que tem como ideia criar um site de respostas a dúvidas frequentes sobre o próprio programa do FrontEnd Mentor. O software tem como base 4 perguntas com imagens ao lado, além de um titulo acima e um fundo de imagem.

O aspecto de interatividade do site se baseia nas imagens ao lado das perguntas, este que por sua vex foi totalmente feito com JavaScript. Tal sistema foi feito com base em que as imagens são originalmente um icone de mais, contudo quando o usuário cilca na imagem esta se torna um icone de menos, revelando então o texto abaixo das perguntas

O código de JavaScript é curto e simples, como seu primeiro passo ele cria uma função recebendo como parâmentros o ID do texto e o ID da imagem, em seguida ele seleciona a imagem e o texto a serem editados com base nos ID's recebidos.

Após isso o ele verifica se a imagem é do simbolo de mais ao averiguar sua fonte, caso a verificação de verdadeiro, o código então transforma a imagem no simbolo de menos analisa qual o ID do texto recebido e, dependendo de qual seja, troca o conteudo da resposta pelo texto correto. Caso a verificação de falso, o código então colocaa fonte da imagem como a do simbolo de mais.
