Descrição

Site criado com base na ideia do app de relacionamentos Tinder, onde cada perfil apresenta uma seleção de cartas, que ficam agrupadas até que o usuário interaja sobre elas, colocando o mouse em cima. Ao colocar o mouse a "mão" de cartes se abre revelando assim todas as cartas contendo as informações do perfil.

Caso o usuário coloque o mouse por cima de qualquer carta o design das mesmas se expande para as laterais facilitanto a visibilidade das cartas, além disso o site permite que o usuário clique no grupo de cartas, expnadindo-o ainda mais para os lados e permitindo uma analise mais intrinseca de cada um dos elementos

A ultima função do site é um sistema de seleção de perfis, feito com 2 botões o Love e o Hate. Estes dois botões quando apertados mandam o perfil na tela para a direita e esquerda respectivamente. Esta ideia iita o sistema de seleção feito no propio Tinder com o arrastar para a direita(match) e esquerda(block). Quando mandado para quaquer um dos lados o grupo de cartas diminue até sumir e o grupo que é enviado em seguida expande até o tamanho normal

A parte de JavaScript do código é composta por 3 funções, sendo que, as funções handleLoveButton e handelHateButton tem a mesma lógica só que com efeitos inertidos. A função de seleciona garante quando o usuário clique no grupo de cartas este se expanda para os lados. Já o funcionamento das funções dos botões é feito de forma que quando o usuário clicar em Love ou Hate o grupo de cartas atual e o que vem a seguir recebem uma classe possibilitando que se movimentem da forma correta e com a animaçã correta
